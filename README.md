# SFAIRP Safety Partners - Industrial Safety Consultancy

A high-performance, responsive web application built for **SFAIRP Safety Partners**, specializing in industrial safety audits, systems, and training.

**Live Project:** [https://www.sfairpsafetypartners.com](https://www.sfairpsafetypartners.com)

---

## Project Overview
This is a professional freelance project designed to establish a high-trust, authoritative online presence for an industrial safety consultancy. The site is optimized for speed, SEO, and lead generation.

### Key Features
- **Modern Tech Stack**: Built with Next.js, Tailwind CSS, and Framer Motion.
- **Server-First Architecture**: Optimized for performance and minimal client-side JavaScript.
- **Advanced SEO**: Fully implemented metadata API and JSON-LD structured data.
- **Responsive Design**: Tailored for all devices, from mobile to desktop.
- **Technical Performance**: Achieves top-tier Lighthouse scores for accessibility and speed.
